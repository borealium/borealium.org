# Divvun Manager

Med Divvun Manager installerer man de fleste verktøy fra UiT (Divvun- og Giellatekno-gruppene). Divvun Manager oppdaterer automatisk installerte verktøy når det kommer ut nye versjoner.

## Installere verktøy

### Alle verktøy for et språk

På framsida av programmet klikker man på det språket man vil ha:

{<figure>
  <img src="/static/images/doc/dm/mac-front.png" />
  <figcaption>Divvun Manager-framsida</figcaption>
</figure>}

Deretter klikker man på __Update__:

{<figure>
  <img src="/static/images/doc/dm/mac-lang-screen.png" />
  <figcaption>Divvun Manager-installeringssida</figcaption>
</figure>}

Du blir bedt om å bekrefte det som lastes ned og installeres:

{<figure>
  <img src="/static/images/doc/dm/mac-installation-confirmation.png" />
  <figcaption>Divvun Manager: bekreft installeringa</figcaption>
</figure>}

Alle pakkene blir lasta ned og installert:

{<figure>
  <img src="/static/images/doc/dm/mac-installation-download.png" />
  <figcaption>Divvun Manager: nedlastinga har starta</figcaption>
</figure>}

Når installeringa er klar bør du starta om att maskina.
